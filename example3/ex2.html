CSS

Now we are using a separate file for css and in this example the file is my_style.css

In order to use the style, we will add to the head section a link to the file with the code
<link rel="stylesheet" href="my_style.css">

Class and id in html

You can use one id only once in your html page, but you can use one class several times in your html page.

In css you have to use symbol # to choose an id. And you have to use symbol . to choose a class.

div and span

You can use div and span to mark some section of your html, in order to format it with css.

The difference between them is that div is a block tag and span is an inline tag.

So the div makes also a line break like this:
This is a second div.

Span does not make a line break like this: This is a second span. This is a third span.
