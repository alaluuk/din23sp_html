CSS

The word CSS means Cascading Style Sheet.

You can write the css-code inside style-block of the head section of your html file.

Or you can create a separate file for the css-code. And then the extension part of the filename is css

div element

You can use div to "choose" some parts of your page. The div does not change the outlook, if you dont't set any style to it.
